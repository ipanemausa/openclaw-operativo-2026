# Loop operativo 0 costo – 2026

Este archivo define el loop básico de trabajo diario para mi ecosistema OpenClaw + Discord + Globy, con foco en costo 0 y control de privacidad.

## 1. Apertura de sesión

- Verificar privacidad y proveedores:
  - Confirmar que GitHub Copilot sigue con:
    - Training desactivado (no usar mis datos para entrenar modelos).
    - Copilot cloud agent desactivado.
- Abrir herramientas de trabajo:
  - PowerShell / terminal.
  - VS Code con este repo `openclaw-operativo-2026`.
  - Paneles principales (OpenClaw, Discord, etc. cuando aplique).

## 2. Trabajo del día

- Escribir al inicio del día:
  - Objetivo del día en 2–3 líneas.
- Registrar decisiones relevantes en este repo:
  - Si son de privacidad / proveedores → actualizar `copilot_politica.md` u otros archivos de políticas.
  - Si son de arquitectura / flujo → añadir secciones aquí o en otros `.md` de `docs/`.

## 3. Cierre de sesión

- Anotar rápidamente:
  - Qué se avanzó hoy.
  - Problemas abiertos que quedan para mañana.
- Verificar que los cambios importantes se hayan guardado y subido (cuando trabaje local):
  - `git status` limpio (sin cambios pendientes).
- Cerrar herramientas:
  - Terminales, VS Code, tabs críticas.

---

Este archivo se irá refinando con el uso diario. No es documentación “perfecta”, es mi loop vivo de trabajo.
